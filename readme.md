## How to start
1. install [nodejs](https://nodejs.org/en/download/)
1. open project folder in terminal
1. run `npm i`

## start devServer
1. run `npm run dev`
1. open http://localhost:5000

## build 4 production
1. run `npm run build`
1. encrease version of css-file  
  `<link rel="stylesheet" href="src/css/style.css?v=4">`
